Pyrrha Menu v1.0.0
Author: rmux

[ Description ]
A multi-purpose utility script for GTA:SA MP (SAMP). Includes weapon, visual, vehicle, and miscellaneous features.

[ Installation ]
1. Install MoonLoader.
2. Ensure you have the required libraries installed (imgui, ffi, memory, vkeys, encoding, lfs).
3. Place 'Pyrrha.lua' into your 'moonloader' folder.
4. (Optional) Place 'tick.mp3' in 'moonloader/resource/pyrrha/' for hitsounds.

[ Default Keybinds ]
U           - Open/Close Menu
F4          - Toggle ESP
F5          - Toggle Lines
F2          - NoReload
F3          - AntiStun
F6          - FakeAFK
F7          - FakeLag
F8          - NoSpread
F9          - GodMode
F10         - WaterDrive
F11         - FireCar
F12         - Instant Crosshair
L.Shift     - Drift Mode (Hold/Toggle)
L.Ctrl      - Speed Boost
O           - Toggle Speed Control
P           - Increase Target Speed
L           - Decrease Target Speed
L.Shift + 0 - Reconnect

[ Features ]
- Weapon: NoSpread, NoReload, Instant Crosshair, Hitsounds.
- Visual: ESP Boxes, Skeleton, Lines, Head Dot, Info Bar.
- Car: Speed Control, Drift Mode, Perfect Handling, Tank Mode, GM Car, Water Drive.
- Misc: AntiStun, FakeAFK, FakeLag, GodMode, Mega Jump, Reconnect Manager.

[ Configuration ]
Settings are saved in 'moonloader/config/Pyrrha/'.
You can create, load, and delete multiple configurations inside the menu.
